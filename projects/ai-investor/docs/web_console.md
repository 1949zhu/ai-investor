## 🎯 什么是 Web 控制台？

这是一个**可视化界面**，让你用浏览器就能看到智能体的一切：

- ✅ 实时系统状态（胜率、置信度、数据量...）
- ✅ 智能体正在做什么
- ✅ 发现的问题和能力缺口
- ✅ 历史决策和绩效
- ✅ 生成的解决方案和改进
- ✅ 实时日志流
- ✅ 手动触发操作

**不再是黑盒命令行，一切尽收眼底！**

---

## 🚀 快速启动

### 方式 1：双击启动（最简单）

找到项目目录下的：
```
启动控制台.bat
```
**双击运行** → 自动安装依赖 → 打开浏览器访问 `http://localhost:5000`

---

### 方式 2：命令行启动

```bash
cd D:\openclaw\workspace\projects\ai-investor

# 启动 Web 服务器
python web_console.py
```

然后打开浏览器访问：**http://localhost:5000**

## ⚙️ 配置选项

编辑 `web_console.py` 底部：

```python
# 修改端口（默认 5000）
app.run(host='0.0.0.0', port=5000, ...)

# 只允许本地访问（更安全）
app.run(host='127.0.0.1', port=5000, ...)
```

## 💡 使用技巧

### 1. 保持浏览器标签页打开
- 数据每 5 秒自动刷新
- 实时看到智能体活动
- 有异常立即发现

### 2. 定期查看"问题与发现"
- 黄色/红色项目需要关注
- 点击"立即分析"让系统找方案
- 查看生成的解决方案文档

### 3. 手动触发检查
- 觉得系统最近不准 → 点"自主进化检查"
- 发现新问题 → 点"自主问题分析"
- 想看看系统状态 → 点"完整自主周期"

### 4. 查看历史报告
- 点击"查看报告"打开报告目录
- 或者直接去 `reports/autonomous/` 文件夹
- 用 Markdown 阅读器打开 `.md` 文件

## ❓ 常见问题

### Q: 为什么指标都是"--"？
A: 第一次运行可能还没数据，点击"刷新数据"或运行一次"完整自主周期"。

### Q: 日志一直显示"等待日志..."？
A: 正常，日志是运行时才会生成。点击任意操作按钮就会看到日志。

### Q: 界面卡顿怎么办？
A: 刷新浏览器页面（F5），或者重启服务器。

### Q: 能在手机上访问吗？
A: 可以！只要手机和电脑在同一 WiFi，用手机浏览器访问 `http://电脑 IP:5000`。
