## 🧠 自主智能体的"思考"过程

### 每次自主周期，系统会问自己：

```
1. "我现在状态怎么样？"
   → 检查胜率、检查模块、检查资源

2. "我缺什么？"
   → 发现能力缺口："我需要龙虎榜数据"
   → 发现问题："新闻 API 失效了"

3. "怎么解决？"
   → 搜索解决方案（用 LLM 分析）
   → 评估方案可行性
   → 生成实现计划

4. "能自己做吗？"
   → 能 → 自己实现
   → 不能 → 生成详细计划，等人帮忙

5. "有进展吗？"
   → 有 → 主动通知人
   → 没有 → 继续尝试
```

```powershell
# 每 4 小时自动运行一次完整周期
$action = New-ScheduledTaskAction -Execute "python" `
  -Argument "D:\openclaw\workspace\projects\ai-investor\autonomous\agent.py" `
  -WorkingDirectory "D:\openclaw\workspace\projects\ai-investor"

$trigger = New-ScheduledTaskTrigger -Once -At (Get-Date) `
  -RepetitionInterval (New-TimeSpan -Hours 4)

Register-ScheduledTask -TaskName "AIInvestor-FullAutonomous" `
  -Action $action `
  -Trigger $trigger `
  -Description "AI 投资系统完整自主周期 - 每 4 小时一次"
```

## 📊 自主智能体在做什么？

### 完整自主周期流程：

```
【阶段 1】自主进化检查
├─ 分析系统现状
├─ 检查决策质量
├─ 检测异常模式
├─ 计算置信度
└─ 生成评估报告

【阶段 2】自主问题分析
├─ 识别问题（数据、性能、模块...）
├─ 识别能力缺口（需要什么新能力）
├─ 搜索解决方案（用 LLM 分析）
├─ 评估优先级
├─ 尝试实现
│  ├─ 能自己做 → 自己实现
│  └─ 不能做 → 生成详细计划
└─ 记录改进

【阶段 3】生成综合报告
├─ 整合两个引擎的结果
├─ 生成完整报告
└─ 必要时通知用户
```
