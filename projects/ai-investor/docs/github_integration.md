# GitHub 集成指南

## 功能

AI 自主管理 GitHub 仓库：
- 监控代码变更
- 自主提交（智能分类）
- 同步 GitHub
- 生成变更日志

---

## 模块

**github_integration.py** - Git 基础操作

**git_agent.py** - AI 自主管理

---

## 使用

```bash
python git_agent.py workflow    # 完整工作流
python git_agent.py status      # 查看状态
python git_agent.py changelog   # 生成日志
```
