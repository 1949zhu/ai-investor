# AI 投资顾问智能体系统

基于 CrewAI 构建的多智能体投资顾问系统，模拟顶级投资团队的决策流程。

## 架构

```
首席投资官 (CIO)
    ├── 宏观分析师
    ├── 量化分析师
    └── 风控官
```

## 安装

```bash
pip install crewai crewai-tools langchain langchain-core chromadb
```

## 使用

```bash
# 运行每日决策
python -m agents.daily_decision

# 运行智能体系统
python -m agents.main

# 查看记忆
python -m agents.memory.view
```

## 智能体角色

### 首席投资官 (CIO)
- 最终决策者
- 协调各分析师意见
- 生成投资建议报告

### 宏观分析师
- 分析宏观经济数据
- 解读政策动向
- 判断市场情绪

### 量化分析师
- 策略回测验证
- 统计分析
- 绩效评估

### 风控官
- 风险评估
- 仓位控制
- 压力测试

## 决策流程

1. 宏观分析师收集市场信息
2. 量化分析师回测验证策略
3. 风控官评估风险
4. CIO 综合决策
5. 生成报告

## 记忆系统

- 短期记忆：当前分析上下文
- 长期记忆：历史决策案例（向量数据库）
- 反思日志：定期复盘

## 风险控制

| 指标 | 限制 |
|------|------|
| 最大回撤 | 15% |
| 单只股票 | 20% |
| 单行业 | 40% |
| 现金下限 | 10% |
| 止损线 | 8% |
| 止盈线 | 20% |

## 输出

每日生成投资决策报告，包含：
- 市场判断
- 操作建议
- 决策理由
- 风险提示
- 置信度
